## Campus Buddy v1.1.0 — 性能优化与加载体验重构版 🚀

此版本重点解决系统在处理大型图谱网络时的渲染阻塞、加载卡顿，并对系统管理员控制台的过渡交互与加载状态进行了大幅度的体验重构与稳定性优化。

### 🌟 新增与改进特性
1. **加载动画引擎重构 & 粒子星图分离**：
   - 彻底将 Canvas 2D 粒子星图数学计算与渲染逻辑从 [LoginLoadingScreen.vue](file:///e:/学习/大二下课程/数据结构与算法/数据结构大作业/Campus-Buddy/frontend/src/components/auth/LoginLoadingScreen.vue) 中剥离出来，封装在独立的 [LoadingParticleEngine.ts](file:///e:/学习/大二下课程/数据结构与算法/数据结构大作业/Campus-Buddy/frontend/src/services/LoadingParticleEngine.ts) 服务中。
   - 优化粒子连线算法，省去内部循环的多余 Map 查找与 `Math.sqrt` 冗余计算，大幅降低 CPU 开销。
   - 修复了加载进度低于 5% 时粒子坐标剧烈跳跃、向心收缩效果异常的问题，让动画收缩过渡更加平滑。

2. **系统管理员登录与仪表盘性能飞跃**：
   - **Vue Watch 性能修复**：消除了对全局 Map 图结构进行 `{ deep: true }` 深度递归监听带来的主线程卡死问题。
   - **聚类系数计算逻辑优化**：通过对兴趣圈子进行哈希缓存和内层逻辑优化，避免了大规模学生图谱结构（1500+ 节点）遍历计算时的帧率骤降。
   - **过渡加载体验重现**：管理员登入时，采用专为管理员定制的加载视图（无杂乱标签、无粒子 CPU 开销，仅保留数码发光百分比与 🤖 头像脉冲动画），以 `#020617` 实色背景过渡，杜绝底层组件穿透泄漏。
   - **延迟渲染架构**：通过延迟 350ms 挂载复杂指标子卡片与计算逻辑，避开 CSS 退出淡出动画的计算峰值，达成 60 FPS 丝滑切换。

3. **CI/CD 自动化持续部署**：
   - 升级 `.github/workflows/ci.yml` 支持自动部署到 GitHub Pages。每次 push 至 master 分支且 CI 成功后，将自动构建并推送到仓库的 `gh-pages` 分支。
